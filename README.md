This is a simple To-Do List desktop application made using **Python** and **Tkinter**.
It helps users add tasks, mark them as completed, and delete tasks using a graphical window.

## STEP 1: Install Python

1. Open your web browser
2. Go to https://www.python.org
3. Click **Download Python**
4. Install Python
5. While installing, make sure to check:
   ‚úî Add Python to PATH

## STEP 2: Check Python Installation

1. Open **Command Prompt** (Windows) or **Terminal** (Mac/Linux)
2. Type the command:

   python --version

3. If Python is installed, you will see the version number

## STEP 4: Open the Project Folder

1. Open the folder where the project is downloaded
2. Make sure the file **main.py** is inside the folder

---

## STEP 5: Run the Application

1. Open Command Prompt / Terminal
2. Go to the project folder:

   cd To-Do-List-App

3. Run the program:

   python main.py

4. The To-Do List window will open

---

## STEP 6: How to Use the Application

### Add a Task
1. Type a task in the input box
2. Click **Add Task**
3. Task will appear as **Pending**

### Mark Task as Completed
1. Select a task from the list
2. Click **Mark as Completed**
3. Status will change to **Completed**

### Delete a Task
1. Select a task from the list
2. Click **Delete Task**
3. Task will be removed

---

## STEP 7: Warning Messages

- If you click a button without selecting a task, a warning appears
- If you try to add an empty task, a warning appears
